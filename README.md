## Contributing

Feel free to improve the plugin and send us a pull request.

We use gulp to compile Less files and minify the JavaScript.
Assuming [nodejs](http://nodejs.org/) is installed, you can run a
build with:

```sh
cd src
npm install
npm install -g bower
bower install
npm install -g gulp
gulp build
```
